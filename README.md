# ecommerce_app
- E-commerce app developed with API integration and state management using Bloc.
- Key features include:
  o  Sign In
  o  Sign Up
  o  View All Products
  o  Product Details
  o  Cart List Screen
  o  Checkout Screen
  o  Order Details
- Payment integration in progress.
- used shared preference to offline store user crediantials.
